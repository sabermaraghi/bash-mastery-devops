# Bash Mastery for DevOps Engineers

A 21-day professional learning path to master Bash scripting.  
Includes daily lessons, CI/CD, and integration with Kubernetes & ArgoCD.

## Structure
- `/scripts/` – Bash scripts by topic
- `/docs/` – Daily tutorials
- `/projects/` – Full automation projects
- `/ci-cd/` – GitHub Actions
- `/k8s/`, `/argocd-apps/` – Ready for production deployments
